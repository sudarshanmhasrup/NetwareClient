## Netware Client
Network Request Library for Kotlin developed by Sudarshan Mhasrup to make network request easier. 💫
### About this project: 
I can understand the pain and complexity of sending an API request from your Kotlin client application to your server. So I decided to solve this problem by developing a client side library. Netware Client is a library developed by myself to make network requests in Kotlin simpler and easier. This library is written in Kotlin, and I'm always open to accepting contributions from other people so that this library can be improved. If you want to suggest any improvements, please let me know about them. Enjoy using this library.
(Note: I'm not yet finished with the development of this library.)
